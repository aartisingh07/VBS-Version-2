# 🏦 VBS Bank – Version 2 (MERN Stack)

Developed by **Aarti Singh**

A modern and responsive **Virtual Banking System** built using the **MERN Stack**.
VBS Bank Version 2 is an upgraded reimagining of an earlier banking project, redesigned with a cleaner UI, modular architecture, responsive dashboard experience, and modern digital banking features.

---

## 🚀 Project Overview

VBS Bank Version 2 simulates a real-world internet banking platform where users can:

* Open a new bank account
* Login securely using Customer ID
* Access a private dashboard
* Transfer money using multiple options
* View transaction history
* Manage debit card settings
* Explore investment products
* Apply for loans with nearby branch support
* Use responsive mobile sidebar navigation
* Manage account controls through interactive profile menu
